# גאיהלנד — מערכת ניהול

מונורפו לכל מערכות גאיהלנד: הזמנות, דשבורד ניהולי, פיננסים, כוח אדם, ואתר.

---

## 🔴 חוק הזהב

**שום דבר לא נשלח בלי שכל הוולידטורים עוברים.**

```bash
./validate-all.sh     # חייב להסתיים ב-✅ ALL VALIDATORS PASSED
```

הסיבה: הכשל החוזר בפרויקט הזה הוא עריכה שמוחקת קוד עובד. הוולידטורים הם חוזה —
כל פיצ'ר שנשלח נרשם בהם, וכל עריכה שמוחקת פיצ'ר נתפסת לפני שהיא מגיעה לייצור.

## תהליך עבודה

📖 **מדריך מלא לפי רכיב:** [`docs/workflow.md`](docs/workflow.md)

```bash
git checkout -b feature/שם-הפיצ׳ר
# ... עריכה ...
./validate-all.sh                    # חייב ✅
git add -A && git commit -m "תיאור"
git push origin feature/שם-הפיצ׳ר
```

**כשמוסיפים פיצ׳ר** — מוסיפים אותו גם לוולידטור הרלוונטי, אחרת אין מה שיגן עליו.

## אבטחה

- **אין סודות בריפו.** סיסמאות ומפתחות יושבים ב-Script Properties של Apps Script.
  להגדרה: `Project Settings → Script Properties → WC_PASS`
- הוולידטור בודק שסודות ידועים לא חזרו לקוד.
- מפתח re-member שדלף בעבר — **צריך רוטציה**.
