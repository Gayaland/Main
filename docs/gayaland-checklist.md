## חלק 2 — טופס ההזמנה (הלקוח)

### 2א. זרימה בסיסית — כרטיס רגיל
- [ ] הכפתור "תיאום הגעה" מופיע **ירוק ומעוצב** (לא שקוף)
- [ ] לחיצה פותחת את הפופ-אפ במסך מלא
- [ ] בחירת **סניף** → נפתחים תאריכים
- [ ] בחירת **תאריך** → נפתחות שעות (סבבים)
- [ ] בחירת **שעה** → מופיע מספר ילדים + סוג כרטיס
- [ ] בחירת מספר ילדים משנה את המחיר נכון
- [ ] הטקסט של הכרטיסייה (ההנחה) **קריא** (כהה, לא בהיר)
- [ ] לחיצה על "המשך לתשלום" → מעביר לנייקס
- [ ] **תשלום מלא** → חוזר לאתר עם הודעת הצלחה
- [ ] ההזמנה מופיעה בדשבורד (טאב "מי הגיע")

### 2ב. כרטיסייה (10 כניסות)
- [ ] בחירת "כרטיסייה" מציגה את המחיר וההנחה
- [ ] השלמת רכישת כרטיסייה עובדת
- [ ] אחרי הרכישה — היתרה נכונה (אם הניקוב האוטומטי פעיל)

### 2ג. מימוש שובר יום הולדת
- [ ] בחירת "יש לי שובר" → נפתח שדה קוד
- [ ] הזנת **שובר תקף** → "השובר תקף! הכניסה ללא תשלום"
- [ ] הזנת **קוד שגוי** → "לא נמצא שובר עם הקוד הזה"
- [ ] **השלמת הזמנה מלאה** עם שובר → הזמנה נרשמת
- [ ] השובר סומן כ**מומש בגיליון** ימי ההולדת (טאב "מומש")
- [ ] ניסיון להשתמש **באותו שובר שוב** → "השובר כבר נוצל"

### 2ד. מקרי קצה
- [ ] סבב **מלא** → לא ניתן לבחור בו / מסומן כתפוס
- [ ] בחירת תאריך שעבר → לא אפשרי
- [ ] סגירת הפופ-אפ (X) והחזרה → הדף במקום הנכון, לא "קפוא"
- [ ] גלילה בתוך הפופ-אפ בנייד → הרקע **לא זז** מאחור

## חלק 4 — הדשבורד (מנהל — אתם)

### 4א. כניסה והרשאות
- [ ] כניסה עם Google (שלכם) → רואים את **כל** הטאבים
- [ ] טאב "מי הגיע" — ההזמנות מקובצות **לפי סבב**
- [ ] כל הזמנה מציגה: שם, טלפון, ילדים, תשלום, **שובר** (אם יש), תוספת
- [ ] טאב אירועים — **אין** יותר כרטיס "אימות תשלומים"

### 4ב. פעולות יומיות
- [ ] סימון "הגיע" על הזמנה → מתעדכן (ירוק)
- [ ] ביטול סימון הגעה → חוזר למצב הקודם
- [ ] מסירת תוספת (☕ קפה) → מסומן כנמסר
- [ ] בורר התקופה ("היום" / תאריך / טווח) → מסנן נכון
- [ ] הזמנות **לא-משולמות** לא מופיעות (רק ששולמו)

### 4ג. תשלומים
- [ ] הזמנה ששולמה מסומנת "✓ שולם ואומת"
- [ ] אם יש הזמנה ששולמה בנייקס אך נכשלה בווקומרס → מזוהה

## חלק 5 — העובדים (צוות)

### 5א. הגדרת עובד
- [ ] בגיליון גוגל → טאב **"צוות"** קיים
- [ ] הוספתם עובד: שם משתמש, סיסמה, שם, הרשאות, פעיל=כן
- [ ] ההרשאות מופרדות בפסיקים: `events,arrivals,coupons,hr`

### 5ב. כניסת עובד
- [ ] במסך ההתחברות של הדשבורד → "כניסת צוות"
- [ ] כניסה עם שם משתמש + סיסמה של העובד → נכנס
- [ ] סיסמה שגויה → "סיסמה שגויה"
- [ ] העובד רואה **רק** את הטאבים שנתתם לו
- [ ] טאב HR (אם נתתם) → **לוח חודשי לצפייה בלבד**, בלי עריכה
- [ ] אין לינק "פתח את המערכת המלאה" לעובד

### 5ג. שליטה
- [ ] שינוי "פעיל" ל-**לא** בגיליון → העובד נחסם בכניסה הבאה
- [ ] הסרת הרשאה (למשל `hr`) → העובד כבר לא רואה את הטאב
- [ ] העובד **לא** רואה: הכנסות, הגדרות, עריכת משמרות

### 5ד. חיבור פעם ביום
- [ ] עובד שנכנס בבוקר → נשאר מחובר עד סוף היום (לא צריך סיסמה שוב)
